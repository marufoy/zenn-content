## TryHackMe（THM）

### なぜTHMから始めるのか

THMはボリュームがあるが、以下の基礎を完全に育ててくれる。

- そもそもペンテストとは何か
- Linuxコマンドの使い方
- ネットワークの基礎知識

「何もわからない状態」から始めても
体系的に学べるのが最大の強みだ。

英語だがGoogle翻訳しながら読めば問題ない。
読んで手を動かすスタイルなので定着しやすく、
自信にもつながる。
![](https://storage.googleapis.com/zenn-user-upload/1f0e416f6b59-20260403.png)
VPNで繋いで実際にマシンを操作できるのも大きい。

### 注意点

月々約**1,200円**かかる。
ただし動画を延々見るよりよほど身につく。


## INEのラボ

THMで基礎が固まったら、INEのラボに移る。

「〇〇というツールを使って列挙する」といった
**試験に直結する実践的な操作**をここで学べる。

### INEの動画は見なくていい

英語で何を言っているか聞き取りづらい上に、
時間がかなり長い。
どうしても気になるなら、わからないところに絞って視聴するとGood。

![](https://storage.googleapis.com/zenn-user-upload/6f9daca929f6-20260403.png)

**ラボだけやれば十分だ。**

## まとめ

| | THM | INE |
|---|---|---|
| 目的 | 基礎固め | 試験直結の実践 |
| 形式 | 読む＋手を動かす | ラボ中心 |
| 言語 | 英語（翻訳しながらOK） | 英語（動画は不要） |
| 費用 | 約1,200円/月 | INEサブスク込み |

## どのくらい時間がかかるか

プログラミングはある程度できるが、
**ネットワークは全然わからない**というタイプは
それなりに時間がかかると思った方がいい。

僕の場合はこうだった。

| | 時間 |
|---|---|
| 平日 | 平均3時間 |
| 休日 | 平均4時間 |
| 期間 | 3ヶ月 |

夢中になって一日中やっていた日もあった。

ただ、これは苦痛ではなかった。
ペンテストは**パズルを解く感覚**に近いし、えぇ！そんなことできるんだ！みたいな驚きもあった。
知識が繋がってきた瞬間の面白さがあるので気づいたら没頭していた。

ネットワークに自信がない人ほど
**焦らず3ヶ月のスパンで考えることをおすすめする。**
