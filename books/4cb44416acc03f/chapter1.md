## どんな資格？
eJPT（eLearnSecurity Junior Penetration Tester）は、サイバーセキュリティ企業のINEが提供する入門レベルのペネトレーションテスト資格だ。
![](https://storage.googleapis.com/zenn-user-upload/6acef5e280e3-20260401.png)
ペネトレーションテストとは、システムやネットワークに擬似的に攻撃を仕掛けてセキュリティの穴を見つける技術のことである。
つまり、許可をもらって「危ないとこないよね？」というのを確かめる人の資格。
いわゆるホワイトハッカーというやつだ。
![](https://storage.googleapis.com/zenn-user-upload/79da5441b1ac-20260401.png)

試験の概要

| 項目 | 内容 |
|------|------|
| 試験形式 | ブラウザ上の仮想環境で実技 |
| 問題数 | 約45問（変更の可能性あり） |
| 制限時間 | 48時間 |
| 合格ライン | 70% |
| 受験料 | $299（5万円ぐらい） |
| 難易度 | 入門〜初級 |
|オープンブック| 可（メモ・ネット参照OK）|


## なぜeJPTを取るのか
セキュリティ資格といえばOSCPが有名だが、難易度が高く初心者には敷居が高い。
eJPTはその最初の一歩として最適だ。
実際に手を動かして攻撃する実技形式なので、座学だけの資格より実践的なスキルが身につく。ホワイトハッカーへの足がかりとして、これ以上ないスタート地点である。
国内ではセキスペ（情報処理安全確保支援士）が有名だ。

しかしセキスペはどちらかといえば防御・マネジメント寄りの国家資格であり、実際に手を動かして攻撃する技術とは少し毛色が異なる。
eJPTは**実際に侵入できるかを問う実技試験**。ペネトレーションテスターとして手を動かしたいなら、eJPTの方が直結している。

## テストの難易度はどのぐらい？
ネットワークの知識がないと覚えることはかなり多い。
CompTIA社が出したロードマップでは以下のような位置付けになっている。
![](https://storage.googleapis.com/zenn-user-upload/d2c22d0d71c9-20260401.png)

初心者向けとは言ったが、実践形式であるがゆえに**ネットワークなどをの知識を知らないと体感としては中の中ぐらいの難易度**になる。
ネットワークの知識がある人は妥当な難易度。

とにかくポートやIPのちがいとか範囲などがわかりにくくなるところ（サブネットマスクとか）。
